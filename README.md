# Sigaretten Biets Schandpaal App — minimale v1 output (issue #462)

Dit is gecontroleerde productoutput voor issue #462: een minimale Android-native v1 proof-case voor review.

## Lokaal openen / beoordelen

1. Open deze repository als Android/Gradle project in Android Studio.
2. Laat Gradle sync uitvoeren.
3. Start de app op een emulator of Android-device.

## Aanwezige v1-functionaliteit

- vaste/eenvoudige collega-lijst (zonder zoekfilter);
- per collega sigaretverzoek registreren via simpele knoppen;
- dagteller per collega;
- totaalteller per collega;
- handmatige dagreset (per collega en als totaalreset);
- directe grote korte Engelse verdict/oneliner.

## Scope- en systeemgrenzen

- opslag is lokaal per Android-apparaat (SharedPreferences);
- geen backend;
- geen sync;
- geen login/auth;
- geen cloud;
- geen externe API;
- geen AI;
- geen fotoherkenning;
- geen deployment;
- geen release.

Deze output is voor handmatige review en is geen productrelease.
